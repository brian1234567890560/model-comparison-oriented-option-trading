# Model Comparison-Oriented Option Trading

This repository presents an academic project on model comparison for option-trading strategies. The work evaluates four different modeling approaches against a SPY/S&P 500 benchmark, with the goal of studying how alternative model designs perform under a common market-reference setting.

## Project Overview

The project began as a mathematics final project and was later organized as a formal research-style paper. Its focus is educational and analytical: it compares model behavior, benchmark performance, and the practical limitations of applying quantitative methods to option-trading decisions.

## Research Focus

- Compare four modeling approaches for option-trading analysis.
- Evaluate model outputs against a SPY/S&P 500 benchmark.
- Study how model choice affects observed trading-performance behavior.
- Present the work in a structured academic format suitable for future refinement.

## Context

This is an early formal research project. The structure and formatting reflect its origin as a class-based academic paper, while the repository is intended to preserve the work and support future improvements in methodology, presentation, and reproducibility.
