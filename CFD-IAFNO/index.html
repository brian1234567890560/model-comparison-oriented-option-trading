CFD + IAFNO Cleanroom Airflow Simulation

A cleanroom airflow project combining OpenFOAM CFD, IAFNO neural operators, and Lagrangian particle tracing.

Project Goal

This project builds a workflow for predicting cleanroom airflow and particle transport. OpenFOAM is used to generate CFD velocity fields, and an IAFNO model is trained as a faster surrogate model for predicting future 3D airflow states.

Workflow

  1. Generate cleanroom CFD data using OpenFOAM.
  2. Export the 3D velocity field into NumPy or PyTorch format.
  3. Train an IAFNO model to predict future velocity fields.
  4. Trace Lagrangian particles through the predicted airflow.
  5. Add a particle-energy loss to reduce unrealistic particle motion.
  6. Visualize airflow, particles, and CAD obstacles.

CFD Field

The main simulated quantity is the 3D velocity field:

u(x, y, z, t) = (u, v, w)

Temperature and humidity fields were removed from the current model because the project now focuses on velocity prediction and particle motion.

IAFNO Model

IAFNO learns a field-to-field mapping. Instead of predicting one point, it predicts the next full 3D velocity field from the previous field. 

u_t → u_{t+1}

Lagrangian Particles

Particles are tracked by directly updating their positions according to the local airflow velocity: 

dx_p/dt = u(x_p, t)

This allows the simulation to study how contamination particles move inside the cleanroom.

Loss Function

The model uses a velocity-field prediction loss plus a particle-energy constraint: 

L_total = L_field + lambda_E L_energy + lambda_s L_smooth

The energy loss helps reduce unrealistic particle wiggle and unstable particle acceleration.

Resolution and Particles

The target setup uses a 42 × 42 × 42 grid and 50 particles. Higher resolution gives more spatial detail, but it also increases memory usage and may cause the kernel to crash.

Obstacle and CAD

CAD geometry can be imported for visualization and obstacle handling. To prevent particles from passing through objects, the particle tracer must include obstacle collision checking or an obstacle mask.

Code Included

  • iafno_model.py — simplified IAFNO model
  • particle_lagrangian_loss.py — particle-energy loss
  • train_iafno.py — training loop
  • openfoam_export_pipeline.py — OpenFOAM export placeholder
  • particle_tracing.py — Lagrangian particle advection
